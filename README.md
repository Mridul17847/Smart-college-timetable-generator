# 🎓 Smart College Timetable Generator

A modern and responsive **College Timetable Generator** built using **HTML, CSS, and JavaScript**.  
This project helps users create slot-based academic timetables similar to university systems with subject allocation, conflict detection, and modern UI design.

---

## 🚀 Features

- ✅ Slot-Based Timetable System
- ✅ Add Subjects, Faculty, and Slots
- ✅ Supports Combined Slots (Example: A11+A12)
- ✅ Automatic Slot Allocation
- ✅ Conflict Detection for Filled Slots
- ✅ Remove Subjects by Subject Name
- ✅ Responsive Design
- ✅ Modern Glassmorphism UI
- ✅ Animated Gradient Background
- ✅ Lunch Break and Evening Slots Included
